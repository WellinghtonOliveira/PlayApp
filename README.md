# PlayApp

Aplicativo desenvolvido com **Electron** com o objetivo de criar uma plataforma para visualização de **filmes e séries**, permitindo ao usuário acompanhar seu progresso de forma automática.

O aplicativo foi projetado para salvar o ponto exato onde o usuário parou em um filme ou episódio de uma série, facilitando a continuidade do consumo de conteúdo sem perda de progresso.

Este projeto foi desenvolvido de forma independente, com foco em estudo, experimentação e prática de desenvolvimento de aplicações desktop utilizando tecnologias web.
