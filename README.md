# 🎬 PlayApp

**PlayApp** é um player de vídeo desktop desenvolvido com **Electron** que organiza automaticamente sua biblioteca de filmes e séries e salva o progresso de reprodução para continuar exatamente de onde você parou.

## 🚀 Como usar

O PlayApp suporta **duas formas de carregar sua biblioteca**:

### 📁 Pasta automática
Na primeira execução o aplicativo cria automaticamente uma pasta chamada `videos`.  
Basta colocar seus filmes ou episódios dentro dela.

### 📂 Selecionar pasta
Também é possível escolher **qualquer pasta do computador** como biblioteca de mídia usando o botão **Selecionar Pasta** dentro do aplicativo.

---

## 📂 Organização dos arquivos

Os vídeos devem estar organizados em **pastas**, onde cada pasta representa um título.

Exemplo:
```
videos/
├── Interestelar (2014)
│   └── filme.mp4
├── Breaking Bad
│   ├── S01E01.mp4
│   └── S01E02.mkv
└── Batman
    └── batman.mp4
```

## ✨ Funcionalidades

- Player de vídeo integrado
- Salvamento automático do progresso
- Continuação de onde parou
- Seleção de pasta personalizada
- Controles de reprodução e volume
- Atalhos de teclado
